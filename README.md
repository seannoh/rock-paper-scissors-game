# Rock Paper Scissors Game

A simple web application in which the user can play rock, paper, scissors against a randomly generated computer opponent. The app uses HTML and CSS for the UI and pure JavaScript for the game logic. It is deployed on Github Pages as linked below.

## Links

- [Website](https://seannoh.github.io/rock-paper-scissors-game/)
- [Repository](https://github.com/seannoh/rock-paper-scissors-game)

## Usage

1. Upon page load, the user is presented with some text and a start button. 
2. Press the start button to start the game.
3. Follow the series of prompts to play the game.
